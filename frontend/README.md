# Full Stack Weather App 🌤

This is a full-stack web application built using:

- Frontend: React + Vite
- Backend: Node.js + Express
- API: OpenWeather API

It allows users to search for a city and view current weather + 5-day forecast.
## 📁 Project Structure
Frontend Setup

cd frontend
npm install
npm run dev

----------------------------------

Backend Setup

cd backend
npm install
node server.js

----------------------------------

Environment Variables

Create a .env file inside backend folder.

Example:

API_KEY=your_openweathermap_api_key
PORT=5000

----------------------------------
